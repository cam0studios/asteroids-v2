## Controls
- <kbd>W</kbd><kbd>A</kbd><kbd>S</kbd><kbd>D</kbd> - Player movement
- <kbd>LMB</kbd> or <kbd>RMB</kbd> (hold) - Shoot
- <kbd>Z</kbd> - Toggle Constant Fire
- <kbd>Space</kbd> - Dash / Dodge
